# 折纸机器人动力学仿真

## 1. 概述

本文档描述 Godot 4.6 物理引擎中折纸机器人动力学仿真的各项参数、编号约定，及其与运动学推导文档 `docs/折纸机器人运动学推导/折纸机器人运动学推导.md` 中数学符号的对应关系。

仿真场景文件：`scenes/origami_robot.tscn`  
运动控制脚本：`scripts/robot_motion.gd`  
浮力脚本：`scripts/buoyancy/buoyancy_mesh.gd`  
外层场景：`scenes/ocean.tscn`

## 2. 坐标系约定

### 2.1 运动学文档坐标系（机体坐标系）

建立以完全展开的机器人中心为原点 $O$：

| 轴 | 方向 | 说明 |
|---|---|---|
| $+X$ | 前方 | — |
| $+Y$ | 右方 | — |
| $+Z$ | 下方 | 右手系，$Z$ 轴向下 |

主折痕状态用单位向量描述：
$$
A_i = \begin{bmatrix}\cos\theta_i\cos\varphi_i, & \cos\theta_i\sin\varphi_i, & \sin\theta_i\end{bmatrix}^T
$$

### 2.2 Godot 仿真坐标系

| 轴 | 方向 | 说明 |
|---|---|---|
| $+X$ | 右方 | — |
| $+Y$ | 上方 | — |
| $+Z$ | 后方 | 左手系，$Y$ 轴向上 |

**SDF → Godot 坐标转换**：
- 位置/顶点：$(x, y, z)_{\text{sdf}} \rightarrow (x, z, -y)_{\text{godot}}$
- 基向量：$B_{\text{godot}} = C \cdot B_{\text{sdf}} \cdot C^T$，其中 $C = \text{Basis}\bigl((1,0,0), (0,0,-1), (0,1,0)\bigr)$

> **注**：运动学推导中的 $+Z$ 向下与 Godot 的 $+Y$ 向上是反的。仿真中所有物理计算（二面角、法线等）均在 **Godot 世界坐标系** 中进行，与运动学推导的结果在数值上对应（仅坐标轴标签不同）。

## 5. 二面角与折叠状态

### 5.1 二面角定义

仿真中的折叠/展开度量是**二面角** $\delta$（相邻面法线绕折痕方向的有符号夹角）：

- $\delta = 0$：两 face **共面**（完全展开）
- $|\delta| > 0$：**折叠**，角度越大折叠越深

计算方式：
$$
\sin\delta = \mathbf{a} \cdot (\mathbf{n}_a \times \mathbf{n}_b), \quad
\cos\delta = \mathbf{n}_a \cdot \mathbf{n}_b, \quad
\delta = \text{atan2}(\sin\delta, \cos\delta)
$$

其中：
- $\mathbf{a}$ = 折痕方向（`HingeJoint3D.basis.z` 归一化）
- $\mathbf{n}_a, \mathbf{n}_b$ = 相邻面片的几何法线（两条折痕的叉积，统一朝外侧）

### 5.2 初始状态（折叠状态）

仿真初始捕获的二面角：

| 关节 | 类型 | 初始 $\delta$ | 说明 |
|------|------|--------------|------|
| `joint0` | $P_1$ | $0.00°$ | 从折痕，共面 |
| `joint1` | $A_1$ | $90.00°$ | 主折痕，完全折叠 |
| `joint2` | $P_2$ | $0.00°$ | 从折痕，共面 |
| `joint3` | $A_2$ | $90.00°$ | 主折痕，完全折叠 |
| `joint4` | $P_3$ | $0.00°$ | 从折痕，共面 |
| `joint5` | $A_3$ | $90.00°$ | 主折痕，完全折叠 |
| `joint6` | $P_4$ | $0.00°$ | 从折痕，共面 |
| `joint7` | $A_4$ | $90.00°$ | 主折痕，完全折叠 |

> 初始状态与运动学约束一致：从折痕两侧共面（$\delta_{P_i} = 0$，对应 $A_i \perp A_{i+1}$），主折痕两侧折叠 $90°$。

### 5.3 与运动学公式的关系

运动学文档给出从折痕二面角：
$$
\cos\delta_{P_i} = 1 - 2(A_i \cdot A_{i+1})
$$

- 当 $A_i \perp A_{i+1}$（$c = 0$，完全展开）：$\cos\delta_{P_i} = 1 \Rightarrow \delta_{P_i} = 0°$ ✓
- 当 $A_i \cdot A_{i+1} = 1/2$（夹角 $60°$）：$\cos\delta_{P_i} = 0 \Rightarrow \delta_{P_i} = 90°$

仿真中初始状态满足 $A_i \perp A_{i+1}$，故 $\delta_{P_i} = 0°$。

## 6. 运动控制参数

### 6.1 控制器导出参数（`robot_motion.gd`）

| 参数名 | 默认值 | 单位 | 说明 |
|--------|--------|------|------|
| `cycle_period` | 6.0 | s | 一个完整折叠循环的周期 |
| `fold_depth` | 1.0 | — | 折叠深度（0=完全展开，1=折叠到初始深度） |
| `position_gain` | 3.0 | rad/s / rad | PD 比例增益 |
| `damping_gain` | 1.0 | rad/s / (rad/s) | PD 微分增益（阻尼） |
| `max_velocity` | 1.5 | rad/s | 马达最大角速度限制 |
| `motor_max_impulse` | 8.0 | — | 马达最大冲量 |
| `auto_capture_delay` | 1.0 | s | 物理稳定等待时间 |

### 6.2 运动规划（一个周期）

```
0.00 ~ 0.30: 停留初始（完全折叠，δ = 90°）
0.30 ~ 0.40: 初始 → 半开（ease-out，δ: 90° → 45°）
0.40 ~ 0.60: 停留半开（δ = 45°）
0.60 ~ 0.70: 半开 → 初始（ease-in，δ: 45° → 90°）
0.70 ~ 1.00: 停留初始（完全折叠，δ = 90°）
```

目标二面角计算：$\delta_{\text{target}} = \delta_{\text{initial}} \times \text{fold\_factor} \times \text{fold\_depth}$

### 6.3 马达控制策略

- **只驱动主折痕**：`joint1, joint3, joint5, joint7` 启用马达
- **从折痕自由跟随**：`joint0, joint2, joint4, joint6` 关闭马达
- **关节同步**：每帧将 `HingeJoint3D` 的 `global_transform` 同步到其 `node_a`（parent body），确保折痕方向随面片旋转更新
- **速度方向校正**：每个关节自动检测马达正速度对二面角的影响方向，必要时取反

## 7. 浮力参数

每个面片挂载 `buoyancy_mesh.gd` 脚本，参数如下：

| 参数名 | 默认值 | 单位 | 说明 |
|--------|--------|------|------|
| `buoyancy_multiplier` | 2.0 | — | 浮力倍率（补偿薄壳外廓排开体积） |
| `water_drag` | 1.5 | — | 线性阻尼系数 |
| `water_angular_drag` | 1.5 | — | 角阻尼系数 |
| `fluid_density` | 1000 | kg/m³ | 水的密度 |

> **当前配置**：所有面片的碰撞形状已禁用（`disabled = true`），面片间无碰撞排斥，纯靠关节约束维持结构。
